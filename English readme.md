JustAuth, as you can see, is just a **a third-party authorized login** toolkit** that allows us to get rid of the cumbersome third-party login SDK and make the login **So easy!* *

Project open source address: [gitee](https://gitee.com/yadong.zhang/JustAuth) | [github](https://github.com/zhangyd-c/JustAuth)    
Project documentation: [Reference Document] (https://docs.justauth.whnb.wang)

## Features

Not much nonsense, just two words:

1. **Full **: More than 10 third-party platforms have been integrated (the basics are commonly used at home and abroad), and there are still expansion plans in the future!
2. ** Jane **: The API is the easiest to design (see `Quick Start` below), try to make you use it without obstacles!

## Quick start

- Introducing dependencies
```xml
<dependency>
    <groupId>me.zhyd.oauth</groupId>
    <artifactId>JustAuth</artifactId>
    <version>1.10.1</version>
</dependency>
```
- Call api
```java
// Create an authorization request
AuthRequest authRequest = new AuthGiteeRequest(AuthConfig.builder()
        .clientId("clientId")
        .clientSecret("clientSecret")
        .redirectUri("redirectUri")
        .build());
// Generate authorization page
authRequest.authorize();
// Authorized login will return code (auth_code (Alipay) only), state, version 1.8.0, you can use AuthCallback class as the parameters of the callback interface
// Note: JustAuth defaults to saving state for 3 minutes. If it is not used within 3 minutes, it will automatically clear the expired state.
authRequest.login(callback);
```
